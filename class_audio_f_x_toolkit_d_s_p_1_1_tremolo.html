Detailed Description

Tremolo

A classic tremolo effect. This is done be modulating the input audio signal by an oscillator. This is known as Amplitude modulation. In this case, the user is given control over the frequency rate and ampitude of the oscillator.

https://en.wikipedia.org/wiki/Tremolo_(electronic_effect)
